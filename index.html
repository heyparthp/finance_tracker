📧 Gmail sync

Auto-pull bank transaction alerts. Works in iOS Safari via Google sign-in.

One-time setup — needs a Google Cloud OAuth Client ID:

  1. Go to console.cloud.google.com → New project
  2. APIs & Services → Enable → Gmail API and Google Sheets API
  3. Credentials → Create → OAuth 2.0 Client ID → Web application
  4. Add this page's URL under "Authorized JavaScript origins"
  5. Paste Client ID below and tap Connect

Alternative: Quick Authorization (No Google Cloud setup)

Generate a temporary access token directly from Google:

  1. Open the Google OAuth 2.0 Playground.
  2. In the left list, select Gmail API v1 and check https://www.googleapis.com/auth/gmail.readonly.
  3. Click Authorize APIs and select your Google account.
  4. Tap Exchange authorization code for tokens.

🤖 Telegram iOS Shortcuts Syncer

iOS workaround: Setup an Apple Automation Shortcut to forward transaction SMS messages to a Telegram Bot, then sync them here.

📋 Sync Setup Instructions:

Step 1: Create Telegram Bot
  1. Search for @BotFather on Telegram. Send /newbot, follow prompts, and copy the HTTP API Token.
  2. Search for @userinfobot and tap Start to get your numeric Chat ID.
  3. CRITICAL: Send a message (e.g. /start) to your newly created bot to authorize it.
Step 2: iOS Shortcuts Automation
  1. Open iOS Shortcuts app → Automation tab → Tap + (or New Automation).
  2. Select "Message" as the trigger:
    • Set Message Contains to: spent, debited, spent on Card, Rs.
    • Select Run Immediately (turn off "Ask Before Running").
    • Tap Next.
  3. Tap New Blank Automation → add action: URL. Paste: https://api.telegram.org/bot<YOUR_BOT_TOKEN>/sendMessage
  4. Add action: Get Contents of URL (connect it to the URL action).
  5. Expand the Get Contents of URL action:
    • Change Method to POST.
    • Headers: add Content-Type: application/json.
    • Request Body: select JSON.
    • Add field chat_id (Type: Text) and paste your numeric Chat ID.
    • Add field text (Type: Text) and select Shortcut Input → Message Body.
  6. Save and test it! SMS alerts will now instantly forward to your Telegram bot and sync.
Step 2: Android Auto-Forwarding App
  1. Install a free SMS forwarding app from Google Play Store (e.g. "SMS to Telegram" by LanRen or "SMS Forwarder").
  2. Open the app and configure the destination:
    • Select Telegram Bot as destination.
    • Paste your Bot Token and Chat ID from Step 1.
  3. Set up filters:
    • Add text rule: Only forward messages containing words like debited, spent, Rs., INR, ₹.
    • Optional: Add sender rules for your specific bank's sender ID.
  4. Enable background service and grant SMS read permissions.
  5. Test it! Any incoming matching SMS will auto-forward to your bot.
