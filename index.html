📧 Gmail sync

Auto-pull bank transaction alerts. Works in iOS Safari via Google sign-in.

One-time setup — needs a Google Cloud OAuth Client ID:

  1. Go to console.cloud.google.com → New project
  2. APIs & Services → Enable → Gmail API and Google Sheets API
  3. Credentials → Create → OAuth 2.0 Client ID → Web application
  4. Add this page's URL under "Authorized JavaScript origins"
  5. Paste Client ID below and tap Connect

Alternative: Quick Authorization (No Google Cloud setup)

Generate a temporary access token directly from Google:

  1. Open the Google OAuth 2.0 Playground.
  2. In the left list, select Gmail API v1 and check https://www.googleapis.com/auth/gmail.readonly.
  3. Click Authorize APIs and select your Google account.
  4. Tap Exchange authorization code for tokens.

🤖 Telegram iOS Shortcuts Syncer

iOS workaround: Setup an Apple Automation Shortcut to forward transaction SMS messages to a Telegram Bot, then sync them here.

📋 Detailed Setup Instructions:

1. Bot & Chat Setup
  1. Open Telegram, search for @BotFather and send /newbot to create a bot. Copy the generated API Token.
  2. Search for @userinfobot and start it to get your numeric Chat ID (e.g. 987654321).
  3. Important: Start a chat with your newly created bot (click Start) so it is authorized to send you alerts.
2. iOS Apple Shortcut Setup
  1. Open the native iOS Shortcuts app and switch to the Automation tab.
  2. Tap + and choose Message as the trigger. Set: Sender to Anyone, Message Contains to common keywords (e.g., debited, spent, transaction), and select Run Immediately.
  3. Tap New Blank Automation and add the action: Get Contents of URL.
  4. Set the URL input to: https://api.telegram.org/bot<API_TOKEN>/sendMessage?chat_id=<CHAT_ID>&text=<Shortcut Input> (replace API_TOKEN and CHAT_ID with your values, and set the text parameter to Shortcut Input -> Message -> Body).
